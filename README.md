# Piper Assessment

Before running a project you should First Download [DotNet6](https://dotnet.microsoft.com/en-us/download/dotnet/6.0) and install it.

This Project is created with WPF.

Install Dependencies

```
dotnet restore
```

## Build and Run the app

```
dotnet build
dotnet run 
```
